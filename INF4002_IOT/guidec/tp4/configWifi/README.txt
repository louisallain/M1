- Ce programme permet de configurer des paramètres de connexion wifi et de connexion à un serveur pour
l'envoi de données de température.

- Pour basculer en mode configuration il faut mettre la pâte 14 de la carte à l'état haut, pour re-basculer
en mode envoi de données il faut mettre cette même pâte à l'état bas.

- En mode configuration, on peut accéder à une interface depuis l'adresse 10.0.0.1. Dans cette interface,
il est possible de configurer le SSID et le mot de passe d'un point d'accès wifi ainsi que l'adresse et le port
du serveur où envoyer les données en mode envoi de données.

- En mode envoi de données, le programme collecte des données de température (fictive pour l'exemple) et les envoi
au serveur préciser dans les paramètres et modifiable en mode configuration.
